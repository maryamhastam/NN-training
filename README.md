# NN-training
Intelligent Spacecraft Trajectory Design for Asteroid Mining.
In this project, a NN was trained on a ssynthetically generated dataset of cluster positions and velocities to predict the requiered velocity change (delta_v) for spacecraft maneuvers between target sites.
It actually designs a path between those traget clusters that was chosen by NN by the constraint of "which cluster is likely to have more sample particles to collect"; meaning the most feasible ones.
